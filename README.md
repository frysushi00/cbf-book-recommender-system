# cbf-book-recommender-system

> Book recommendation system using content-based filtering.

![Python](https://img.shields.io/badge/Python-3776AB.svg?style=for-the-badge&logo=python&logoColor=white) ![precision](https://badgen.net/badge/Precision@k/0.93/green) ![mrr](https://badgen.net/badge/MRR/1.0/yellow)

## 📝 Description

A book recommendation engine using content-based filtering that aims to solve the problem of personalizing book suggestions by analyzing attributes of books and matching them against user profiles. Started from loading book data, processing features, calculating similarity metrics, and outputting ranked recommendations based on content similarity.

## ✨ Key Features

- **📚 Content-Based Filtering** — Generates tailored recommendations by computing similarity scores between book attributes.
- **📌 LLM Integration** — Using LLM to generate book keywords for better recommendations.
- **📓 Attribute-Based Matching** — Analyzes intrinsic book properties to suggest relevant books to readers.
